# Salary Data Aggregator

Этот проект представляет собой решение тестового задания для junior Python developer. 
Задача состоит в разработке алгоритма агрегации статистических данных о заработной плате сотрудников в заданные временные промежутки.
Временные гранулярности включают часы, дни и месяцы. Кроме того, в рамках проекта необходимо создать Telegram-бота, 
который будет принимать JSON-запросы с параметрами агрегации и возвращать агрегированные данные в ответ. 
Проект также включает использование MongoDB для хранения данных.
